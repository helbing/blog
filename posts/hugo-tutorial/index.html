Hugo 入门教程

2020.07.12

第 N 次 Hugo 入门，这次重新捡回 Hugo 主要原因是打算使用 org-mode 来管理我的笔记与博文，那么 ox-hugo 自然是不可或缺的，所以重新写了篇 Hugo 入门教程。如果想深入了解 Hugo 的话，推荐看它的官方文档

安装

# macOS
brew install hugo

# archlinux
sudo pacman -Syu hugo

其他的安装方式推荐看官方文档，Install Hugo

快速入门

创建站点

hugo new site /path/to/blog

安装主题

单单创建了站点还是不能够使用的，还需要安装一个主题，这里推荐我使用的主题，zozo

cd /path/to/blog
git clone https://github.com/varkai/hugo-theme-zozo themes/zozo

zozo 主题同时还需要去覆盖配置文件

cp themes/zozo/exampleSite/config.toml .

为了方便后面更新主题，所以最好是使用 Git Submodules 来管理主题，这样可以方便后续更新主题

touch .gitmodules

配置如下

[submodule "theme"]
path=themes/zozo
url=git://github.com/varkai/hugo-theme-zozo.git

配置好后，记得初始化

git submodule init
git submodule update --init --recursive

更新 submodule 代码

git submodule update [--remote]

创建一篇文章

hugo new posts/hello-world.md

Hugo 会自动生成 content/posts/hello-world.md

---
title: "Hello World"
date: 2020-07-12T20:49:42+08:00
draft: true
---

其中 draft 表示为草稿

本地浏览

hugo server

接下来在浏览器打开 localhost:1313 即可，不过由于 hello-word.md 是草稿，所以要用下面的命令

hugo server --buildDrafts
# or
hugo server -D

编译

hugo

编译输出的静态 html 文件，默认保存到 public 目录下

ox-hugo

由于我的计划是使用 org-mode 来写博客，所以我需要使用 ox-hugo 转为 markdown

安装 ox-hugo

我使用的是 use-package 来做包管理，所以只需要加入

(use-package ox-hugo
  :init
  (with-eval-after-load 'ox
    (require 'ox-hugo)))

如果你使用的是其他的包管理，可以参考官方文档来安装，https://ox-hugo.scripter.co/doc/installation/

为 org-mode 添加 snippet

# -*- mode: snippet -*-
# name: create blog
# key: cb
# --
#+HUGO_BASE_DIR: /path/to/blog
#+HUGO_SECTION: posts
#+TITLE: $1
#+DATE: $2
#+HUGO_TAGS: $3
#+HUGO_AUTO_SET_LASTMOD: t
#+HUGO_DRAFT: nil
#+OPTIONS: AUTHOR:nil

$4

<!--more-->

$0

使用上面的 snippet 生成模板，然后编辑博文即可
