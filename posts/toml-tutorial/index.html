TOML 入门教程

2020.07.13

TOML 的全称是 Tom’s Obvious Minimal Language，因为它的作者是 GitHub 联合创始人 Tom Preston-Werner。TOML 的目的是成为一个极简的配置文件格式，可以无歧义地被映射为 hasmap，从而被多种语言解析

格式约定

  • TOML 是大小写敏感的
  • 编码格式使用 UTF-8
  • 使用 # 进行注释
  • boolean 类型使用 true/false，小写格式
  • 时间日期遵循 rfc3339

语法

string

single

key = "value"

multi

key = """
value1
value2
"""

等同于

key = "value1\r\nvalue2\r\nvalue3"

如果不需要换行，则

key = """
value1\
value2
"""

等同于

key = "value1value2"

array

key = [
  "value1"
  "value2"
]

记住，每个数组元素的数据类型必须一致

hashmap

[table]
key = "value"

等同于

{"table":{"key":"value"}}

hashmap 还能嵌套 hashmap

[first.second]
key = "value"

等同于

{"first":{"second":{"key":"value"}}}

hashmap array

[[demo]]
key = "value"

[[demo]]
key = "value"

等同于

[{"key":"value"},{"key":"value"}]
