C 语言编译顺序

23 September 2019
  1. 预处理：宏定义展开，头文件展开，条件编译等，同时将代码中的注释删除，这里并不会检查语法
  2. 编译：检查语法，将预处理后文件编译生成汇编文件
  3. 汇编：将汇编文件生成目标文件(二进制文件)
  4. 链接：C 语言写的程序是需要依赖各种库的，所以编译之后还需要把库链接到最终的可执行程序中去

使用 clang 进行编译

  1. 预处理：clang -E main.c -o main.i
  2. 编译：clang -S main.i -o main.s
  3. 汇编：clang -c main.s -o main.o
  4. 链接：clang main.o -o main
  5. 一步编译：clang main.c -o main

查找程序所依赖的动态库

# Linux
ldd main
# macOS
otool -L main

其他 clang 参数

  • -O 优化选项，0-3，数字越大优化级别越高
  • -g 用于 gdb，lldb 调试，没这个参数无法进行调试
  • -D 指定宏编译，如 -D DEBUG
  • -I 追加头文件的检索路径（可以是绝对路径或相对路径） -I/usr/local/include
  • -L 指定库路径（可以是绝对路径或相对路径）
  • -l 指定库名，通常是 libxxx.so
  • -Wall 显示更多错误信息
  • -lstdc++ 编译 c++ 代码
