Gitbook 极简教程

23 July 2019

gitbook 就是一个电子书编写工具，它会将你编写的 markdown 文章编译转成本电子书

安装

yarn global add gitbook-cli

快速入门

首先通过下面命令初始化一个 gitbook 项目

cd /path/to
mkdir book-name && cd book-name
gitbook init

初始化完成后会生成两个文件，分别是 README.md 和 SUMMARY.md。一个是说明文件，一个是目录文件

之后我们只需要编写 SUMMARY.md 来确立目录结构就行了，如

# Summary

-   [Introduction](README.md)

# Chapter 1

-   [lesson 1](Chapter1/1.md)
-   [lesson 1](Chapter1/2.md)

对应的目录结构如下

|- Introduction
|-|- README.md
|- Chapter 1
|-|- 1.md
|-|- 2.md

接下来执行 gitbook serve 就可以预览 gitbook 了

book.json

{
    "title": "kubernetes 极简教程",
    "plugins": ["katex"]
}

这里只介绍常用的，不常用的可以看这里 https://gitbookio.gitbooks.io/documentation/format/configuration.html

插件推荐

你可以在 https://www.npmjs.com/ 上找有哪些 gitbook 插件，通过关键词 gitbook-plugin- 就能找到 gitbook 插件。这里会推荐一些比较好用的插件

  1. katex
  2. github
  3. search-plus
  4. copy-code-button)

隐藏左下角的 Published with Gitbook

创建样式表文件 styles/website.css

.gitbook-link {
    display: none !important;
}

编辑 book.json 文件

{
    "styles": {
        "website": "styles/website.css"
    }
}
